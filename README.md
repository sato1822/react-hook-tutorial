## 概要
「react hooks」の学習

## 特徴
- ** useState **: 値の監視
- ** useEffect **: 副作用の処理（DOMの書き換え、変数代入、API通信などUI構築以外の処理）を関数コンポーネントで扱えます。

## インストール
1. リポジトリのクローン🚀  
```git clone　 https://github.com/sato1822/react-hook-tutorial.git```

2. 環境設定⚙️  
``` npm i ```&nbsp;or&nbsp;``` npm install ```

3. ブラウザ表示💡  
``` npm run dev ```
